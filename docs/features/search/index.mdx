# 検索

現在はプロフィールの検索ができるようになっており、今後ポストの検索もできるようになる予定です。

また、検索ページにアクセスしたときはおすすめのプロフィールが見られるようになっています。

## プロフィール検索

プロフィール検索では、プロフィールのアットネームや名前、自己紹介文をキーワードで検索できます。

半角スペースでキーワードを区切ると、それぞれのキーワードが含まれるプロフィールを検索します。

## おすすめプロフィール

自分に関連しそうなMewst上のプロフィールを表示しています。

以下の条件に合致したプロフィールが表示されます。

1. 自分がフォローしている人がフォローしている人
2. 1の人がフォローしている人

図にすると以下のようになり、赤枠に囲まれた部分が対象となります。

<img src={Image} alt="" width="500px" />

`プロフィールC` や `プロフィールD` の人だけ表示すると自分が誰をフォローしているのか他の人にわかってしまう可能性があるため、 `プロフィールE` や `プロフィールF` の人まで表示してぼかしています。
