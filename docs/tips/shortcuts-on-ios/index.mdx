# Mewstにポストするアクションを追加する (iOS)

iOSのショートカットアプリを使うと、Safariなどで見ているページのURLをMewstにポストするアクションが追加できます。

「ショートカット」アプリを起動し、以下のように設定してください。

<img src={Image1} alt="" width="500px" />

https://mewst.com/new?content=Hello というURLを開くと、Mewstに「Hello」という内容でポストする画面が表示されます。
このURLを活用してショートカットを作成しています。

上記のショートカットを作成すると、共有メニューからMewstにポストするアクションが選択できるようになります。

Instagramなどの投稿もポストできるようになるので、写真などをポストする際にも便利です。
